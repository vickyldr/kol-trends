# 每周怎么用（本地·双击版）

目标：每周一**双击一下**，自动抓好各国 Hashtag 前十，然后去 Claude 出周报。
用你电脑的真 Chrome + 家里网络，风险最低。

---

## 第一次：装一次（5 分钟，只做一次）

1. **装 Node.js**（如果还没装）：去 https://nodejs.org → 下载 **LTS** 版 → 一路下一步装好。
2. **拿到这个项目文件夹**：
   - 最简单：在 GitHub 仓库页点 **Code → Download ZIP**，解压到桌面。
   - （会用 git 的话 `git clone` 也行。）

就这两步。

---

## 每周一：双击运行

- **Windows**：双击 `tiktok-trends\run-windows.bat`
- **Mac**：双击 `tiktok-trends/run-mac.command`

它会自动：建本周文件夹 → 打开浏览器抓各国前十 → 把结果**复制到剪贴板**。

### ⚠️ 只有第一次：要登录一次

第一次跑会弹出一个浏览器窗口，黑框里提示你登录：

1. 在弹出的浏览器里，点右上角 **Log in**，登录 Creative Center。
   - **用一个单独的 business 账号，别用你发视频的主账号**（万一被风控，主账号不受影响）。
2. 看到 Hashtag 榜能正常显示一长串后，回到黑框，按 **回车**。
3. 登录会被记住，**以后再跑就不用登了**。

---

## 抓完之后：出周报

跑完后，本周的热点词已经**在剪贴板里**了。打开 Claude（这个对话），直接 **Ctrl+V / Cmd+V 粘贴**，说一句：

> 出本周周报

Claude 就会按规则生成挖掘表 + `ideas.csv`（导入飞书用）。

---

## 想调什么（都在 `config/countries.json`）

- 看哪些国家：改 `countries`。
- 前几名 / 几天：改 `top_n`、`period_days`（7/30/120）。

---

## 出问题了

- **抓到的某国是空的**：把那国手动补几个进 `weeks/本周/input-keywords.md`，或直接告诉 Claude。
- **报错 / 抓不到**：把黑框里的报错截图发给 Claude，我来修（多半是 TikTok 改了页面）。
- **不想折腾**：直接照旧手动看榜，把词发给 Claude 也行——分析那部分照样全自动。
