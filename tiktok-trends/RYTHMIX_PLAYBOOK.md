# Rythmix 蹭点 Playbook

> 这是整套流程的"大脑"：给定一个 TikTok 热点，怎么判断它能不能蹭、用哪种方式蹭、怎么不踩版权。
> AI 出周报时严格按本文件执行；人复核时也照这个标准看。

Rythmix 的核心是 **AI 音乐**（文生音乐 / remix 换曲风 / AI MV）。蹭热点的本质是：
**把当下大家在 TikTok 上正在看/正在玩的东西，套进 Rythmix 能产出的内容形态里。**

---

## 一、五种蹭法（产出形态）

| # | 形态 | 一句话 | 适合的热点类型 |
|---|------|--------|---------------|
| 1 | **IP 画面 AI MV** | 把正在火的 IP（游戏/电影/剧/动漫）的"画面感"用 AI 复现成一支 MV | 某个 IP 突然爆火 |
| 2 | **大字报版式** | 左边一个人在某场景里唱歌，右边滚动歌词板，歌词=Rythmix 推广词 | 几乎万能，可叠加到 1/3/4 |
| 3 | **七分相似人物 MV** | AI 生成一个和热点人物七分像的形象，做对口型 MV / 大字报 | 出现某个火的人物/角色 |
| 4 | **舞蹈挑战 AI 翻跳** | 用 AI 驱动直接"抄"一个正在火的舞蹈挑战，作为我们的 AI MV | dance challenge 类 |
| 5 | **热点话题口播 + AI 音乐** | KOL 口播把热点和 AI 音乐结合：文生音乐 / remix 换曲风 / MV，哪种都行 | 话题/梗/事件类 |
| 6 | **热歌音色克隆** | 拿正在火的歌，用 Rythmix 做一层**音色克隆**（换个音色/声线重唱），套翻唱版式 | 某首歌 / 某个歌手上热点 |

> **MV 的本质**：就是"某个画面里某个人对口型唱歌"。所以 1/2/3 经常是同一支视频的不同侧重。
> **大字报（#2）几乎可以叠加到任何一种**——它是版式，不是题材。
> **热歌就直接做 #6 音色克隆**：哪个歌手/哪首歌上热点，直接拿来做一层音色克隆版式——这是热歌类最快的蹭法。

### 对应挖掘表的两层（重要）

挖掘表里要分清两层，别混：

- **主推功能点**（挖掘表「主推功能点」列）= Rythmix 的**产品能力**，只在这四个里选（可多选）：
  `文生音乐` · `AI mv` · `AI驱动` · `音色克隆`
- **蹭法 / 版式要素**（写进挖掘表「修改要点及数量」列）= 怎么把热点改造成我们的：
  `大字报` · `七分相似人物` · `IP氛围复现` · `对口型` · `舞蹈翻跳借鉴` · `原版vs克隆对比` 等。

六种蹭法 → 字段映射：

| 蹭法 | 主推功能点 | 修改要点里常用要素 |
|------|-----------|-------------------|
| #1 IP 画面 AI MV | AI mv | IP氛围复现 + 大字报 |
| #2 大字报版式 | （叠加，看主体） | 大字报（左唱歌右滚动歌词=卖点） |
| #3 七分相似人物 MV | AI mv（+音色克隆） | 七分相似人物 + 对口型 |
| #4 舞蹈挑战翻跳 | AI驱动 | 舞蹈翻跳借鉴 |
| #5 话题口播 + AI 音乐 | 文生音乐 | 口播脚本 + 现场出曲 |
| #6 热歌音色克隆 | 音色克隆 | 原版vs克隆对比 + 大字报 |

## 二、从热点到方案：判断流程

对榜单里每一个词，按顺序走：

1. **翻译 + 查证**：这词字面什么意思？看不懂就联网搜，确认它在 TikTok 上**具体在火什么**
   （一个 IP？一个人？一支舞？一个梗/事件？一首歌？）→ 写成 **idea描述**。
2. **归类**：落到上面六种蹭法里的哪一种（可多选）→ 定 **主推功能点**。
3. **出蹭法**：怎么改造蹭（MV 借鉴哪里、大字报要素、音色克隆方案、做几条）→ 写成 **修改要点及数量**；定 **音频风格**。
4. **找参考视频**：联网搜一条单条样板 → 填 **视频链接**。
5. **版权红线检查**（见第三节）：标 🟢 安全 / 🟡 注意 / 🔴 高风险。
6. **打优先级**：结合"热度 × 和 Rythmix 的契合度 × 版权安全度"给 P0/P1/P2。

### 各类型的标准蹭法

- **IP 爆火** → #1：**不直接出现 IP 形象**，而是复现它的"氛围/画面语言"（配色、场景、镜头感），
  配 AI 原创音乐。叠 #2 大字报，歌词写 Rythmix 卖点。
- **人物爆火** → #3：AI 做七分相似形象（不是本人），对口型唱 Rythmix demo 曲；可做成大字报。
- **舞蹈挑战** → #4：AI 驱动复刻舞步，背景音用 Rythmix 生成/remix 的版本，挂我们的曲。
- **话题/梗/事件** → #5：KOL 口播把梗接到"我用 AI 做了首歌/把它 remix 成 XX 风"，现场出音乐。
- **某首歌火 / 某歌手上热点** → #6 音色克隆（首选）或 #5 remix：
  - **#6 音色克隆**：用 Rythmix 把热歌换个**音色/声线**重唱（如换成不同性别/风格的 AI 音色），套"原版 vs 我翻唱版"对比版式。卖点："任何歌换任何音色"。
  - **#5 remix**：换曲风（lo-fi/phonk/orchestral…），强调"AI 一键换曲风"。
  - ⚠️ 两者都用 Rythmix **自生成的音轨**，不碰原曲母带；音色克隆别克隆到可辨认是某真人歌手本人，见下。

## 三、版权红线（🔴 必须遵守）

视频里**不能直接出现 IP 形象或未授权的内容（音频 + 画面都算）**。具体：

**不可以**
- ❌ 直接放 IP 的官方画面、角色形象、logo、海报、截图。
- ❌ 用未授权的原曲音频（包括完整片段）。哪怕是"火的那首歌"，直接搬原音 = 风险。
- ❌ 用真人明星的肖像/换脸到可被认出是本人的程度。
- ❌ 抄袭某条具体视频的完整画面（构图可借鉴，整条复刻不行）。

**可以**
- ✅ **复现"氛围/风格"**：相似的配色、场景类型、镜头语言、服化道感觉——但不是那个 IP 本身。
- ✅ **七分相似人物**：明显是"致敬/像"，不是冒充本人（避免误导观众是本人）。
- ✅ **AI 原创音乐 / 对原曲做 remix 换曲风**——用 Rythmix 自己生成的音轨，不用原始母带。
- ✅ **借鉴版式/玩法**：大字报版式、舞蹈动作（动作本身一般不受版权保护，但具体编舞可能有争议，谨慎）。

**音色克隆（#6）的版权要点**（单列，因为它最容易踩线）
- ❌ 不用原曲母带/原唱人声；翻唱是用 Rythmix 重新生成的人声 + 伴奏。
- ❌ 不克隆到"明显就是某真人歌手本人声音"的程度（声音权 / 平台规则）。用通用/原创 AI 音色，或明显风格化、转化性的音色。
- ⚠️ 翻唱仍涉及**词曲版权**（旋律+歌词）：短促宣传片是灰区，正式发布/商用要走机械授权。clip 尽量短、转化性强。
- → 默认 🟡；克隆可辨认真人音色 = 🔴。

**风险分级**
- 🟢 安全：纯 AI 原创音乐 + 原创/氛围画面 + 大字报。
- 🟡 注意：七分相似人物、remix 换曲风、音色克隆（通用音色）、复现某 IP 氛围——做得过像就滑向 🔴，需人工把关。
- 🔴 高风险：直接出现 IP 形象/原曲原音/可辨认真人（含克隆真人音色）——**不做**，或必须先拿授权。

> 周报里每个方案都要带分级；🔴 一律不进执行清单，🟡 标"需人工复核"。

## 四、输出字段（对齐挖掘表「0 总填写表」）

每个被选中的热点 = 挖掘表里的一行。只产出**分析阶段**该填的字段，执行/运营字段
（设计直用、KOL选用、Fiverr采买、缩略图、视频附件、填写人、竞品、渠道、各类进度、时间）
落地时再填，**这里不输出**。

| 字段 | 怎么填 |
|------|--------|
| **来源国家** | 这个热点来自哪个市场（榜单里它出现在哪国） |
| **投放地区** | 建议投哪些地区（同热点跨国通用就多选） |
| **idea描述** | ① 这是个热点 → ② 是什么热点（IP/人物/舞蹈/话题梗/歌曲）→ ③ **热点在意什么**（大家为什么爱看、在玩/在追的那个点） |
| **主推功能点** | 只在 `文生音乐 / AI mv / AI驱动 / 音色克隆` 里选（可多选） |
| **修改要点及数量** | **我们怎么蹭**，写具体：MV **借鉴哪里**（画面/场景/镜头）· 大字报**用哪几个要素** · 歌曲走**音色克隆**（克隆成什么音色/对比版式）· 建议做几条（数量/变体） |
| **音频风格** | 曲风或音色方向（如 phonk / lo-fi / city-pop / 女声转男声…） |
| **视频链接** | 🎬 **参考视频（必填）**：一条能点开的 TT/YT/IG 单条样板视频；找不到才退到 trend/discover 聚合页 |
| **版权**（附加列） | 🟢/🟡/🔴 + 一句注意事项（挖掘表本身没有，但我们保留做初筛把关） |

> idea描述 与修改要点的区别：**idea描述只讲热点本身**（是什么、火在哪）；
> **修改要点只讲我们怎么改造蹭**（借鉴点、版式要素、克隆方案、数量）。别写串。

---

## 五、优先级建议

- **P0**：热度高 + 契合度高 + 🟢/🟡可控 → 本周就做。
- **P1**：契合度高但热度中等，或需要点制作量 → 排期。
- **P2**：能蹭但弱，或 🟡 偏风险 → 备选/观察。
- 🔴 一律不进。
